# Digital ASIC Design

This repository contains a series of examples and labs for learning ASIC design using Open source tools, from RTL to GDS.  The material aims to support the VLSI learning process. Most of the material has been prepared by Dr. Juan David Guerrero Balaguera. It might contain material from other open sources.

The resources in this repository have been created in January 2026 using the current state of the Open Source tools availabe at the moment. There is no waranty of future updates at the same peace that the open source tools evolve.
